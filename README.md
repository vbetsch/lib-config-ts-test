[@vbetsch/lib-config-ts](https://github.com/vbetsch/lib-config-ts)

## Install or update

```bash
npm install github:vbetsch/lib-config-ts#DEV_BRANCH
```

## Run

```bash
npm run start
npm run eslint
```

## Requirements

### dependencies

* You just need your runner (tsx for example).
* You have to install **@types/jest** if you want to write Jest tests.

## Modules

### tsconfig

1. **strict**:
    * _No dependencies needed_
2. **jsx**:
    * @types/react
