# lib-config-ts-test

[@vbetsch/lib-config-ts](https://github.com/vbetsch/lib-config-ts)

## Install or update

```bash
npm install -D @vbetsch/config-eslint@latest
npm install -D @vbetsch/config-jest@latest
npm install -D @vbetsch/config-prettier@latest
npm install -D @vbetsch/config-tsconfig@latest
```

## Testing

Run npm commands in this order :

```bash
npm run start:dev
npm run prettier
npm run jest
npm run tsconfig
npm run eslint
npm run tsconfig:build
npm run start:prod
```
