# lib-config-ts-test

[@vbetsch/lib-config-ts](https://github.com/vbetsch/lib-config-ts)

## Install or update

```bash
npm install @vbetsch/config-eslint
npm install @vbetsch/config-jest
npm install @vbetsch/config-prettier
npm install @vbetsch/config-tsconfig
```

## Testing

Run npm commands in this order :

```bash
npm run start:dev
npm run prettier
npm run jest
npm run tsconfig
npm run eslint
npm run tsconfig:build
npm run start:prod
```
