[@vbetsch/lib-config-ts](https://github.com/vbetsch/lib-config-ts)

## Install or update

```bash
npm install github:vbetsch/lib-config-ts#DEV_BRANCH
```

## Run

```bash
npm run start
npm run eslint
```

## Testing

Run npm commands in this order :
```bash
npm run start:dev
npm run prettier
npm run jest
npm run tsconfig
npm run eslint
npm run tsconfig:build
npm run start:prod
```
